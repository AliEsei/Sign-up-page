# Sign-Up Page

## Overview
This project is a modern, responsive **Sign-Up page** built using **HTML, CSS, and JavaScript**. It provides users with a visually appealing form and real-time validation to ensure input quality and password strength. The design emphasizes usability and clean aesthetics.

## Features
- Responsive layout with a **two-panel design** (visual left side and form right side)
 - Interactive **form validation** for:
  - Username (3â€“15 characters)
  - Full name (requires first and last name)
  - Email (valid email format)
  - Password (strength validation and forbidden words check)
- Real-time **password strength feedback** (Weak, Medium, Strong)
- Visual feedback for errors (red) and valid input (green)
- Dynamic **Create Account button** activation based on validation
- Clear **success message** on form submission
- Professional typography using **Google Fonts (Figtree)**
- Custom color palette and gradients for modern UI look
